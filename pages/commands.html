Liste des commandes :

(Prefix = w!)

  • Commandes de modération
  • Commandes pour les salons privés
  • Commandes pour les mini-jeux
  • Commandes memes

  • Commandes de modération :

  • w!setup : Vous permet d'activer ou désactiver les fonctionnalités de Winro
  • w!mute : Permet de réduire au silence un membre
  • w!unmute : Permet de restaurer la voix à un membre
  • w!tempmute : Permet de réduire au silence un membre pendant un temps défini
  • w!kick : Permet d'expulser un membre
  • w!ban : Permet de bannir un membre
  • w!tempban : Permet de bannir un membre pour un certain nombre de jours
  • w!clear : Permet de supprimer le nombre de messages indiqué
  • w!warn : Permet d'avertir un membre
  • w!warns : Permet de voir les avertissements d'un membre
  • w!clearwarns : Permet de supprimer tous les avertissements d'un membre

  • Commandes pour les salons privés :

  • w!rdel : Permet de forcer la suppression d'un salon privé
  • w!create : Permet de créer un salon textuel privé
  • w!add : Permet d'ajouter des membres à votre salon privé
  • w!quit : Permet de quitter un salon privé

  • Commandes mini-jeux :

  • Winro New Horizons :

  • w!island : Donne une page d'aide pour le mini-jeu Winro New Horizons
  • w!shop : Donne une page avec des objets achetables
  • w!inv : Permet de voir le contenu de son inventaire
  • w!buy : Permet d'acheter des objets présents dans la boutique
  • w!xpsell : Permet de vendre un des objets présent dans son inventaire
  • w!picking : Permet de récolter des fruits toutes les heures
  • w!fishing : Permet de pêcher des poissons toutes les heures
  • w!delisland : Permet de supprimer son île et son inventaire

  • Commandes memes :

  • w!toz : Essaye toi même
  • w!arouf : J'ai vraiment besoin de t'expliquer ?
  • w!pertedetemps : Phillipe Etchebest MVP
  • w!dorime : Voilà d'où je puise toute mon inspiration

  • Autres commandes :

  • w!help : Donne la liste de toutes les commandes de Winro
  • w!say : Permet au bot d'envoyer votre message
  • w!ping : Permet de vérifier si le bot fonctionne
  • w!winro : Donne une petite description du bot

Si vous rencontrez un problème avec l'une de ces commandes, n'hésitez pas à le signaler sur le discord de support !
