Liste des commandes :

(Prefix = w!)

  • Commandes de modération
  • Commandes pour les salons privés
  • Commandes memes

  • Commandes de modération :

  • w!admin : Vous affiche la liste des commandes de modération
  • w!logs : Permet de créer un salon dédié aux logs de Winro
  • w!clear : Supprime le nombre de messages indiqué
  • w!warn : Permet d'avertir un membre [Bêta]
  • w!warnings : Permet de voir les avertissements un membre [Bêta]
  • w!cwarn : Permet de supprimer tous les avertissements d'un membre [Bêta]
  • w!msetup : Permet de setup le mute sur votre serveur
  • w!tempmute : Permet de réduire au silence un membre pendant un temps défini
  • w!mute : Permet de réduire au silence un membre
  • w!unmute : Permet de restaurer la voix à un membre
  • w!kick : Permet d'expulser un membre
  • w!ban : Permet de bannir un membre
  • w!tempban : Permet de bannir un membre pour un certain nombre de jours

  • Commandes pour les salons privés :

  • w!rooms : Donne la liste des commandes pour les salons privés [Bêta]
  • w!rsetup : Permet de setup les salons privés sur votre serveur
  • w!rdel : Permet de forcer la suppression d'un salon privé
  • w!create : Permet de créer un salon textuel privé
  • w!add : Permet d'ajouter des membres à votre salon privé
  • w!quit : Permet de quitter un salon privé

  • Commandes memes :

  • w!memes : Donne la liste des commandes de memes
  • w!toz : Essaye toi même
  • w!arouf : J'ai vraiment besoin de t'expliquer ?
  • w!pertedetemps : Phillipe Etchebest MVP
  • w!dorime : Voilà d'où je puise toute mon inspiration

  • Autres commandes :

  • w!say : Permet au bot d'envoyer votre message
  • w!ping : Permet de vérifier si le bot fonctionne
  • w!me : Donne une petite description du bot
